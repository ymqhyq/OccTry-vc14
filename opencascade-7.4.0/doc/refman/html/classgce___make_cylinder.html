Detailed Description

This class implements the following algorithms used to create a Cylinder from gp.

  • Create a Cylinder coaxial to another and passing through a point.
  • Create a Cylinder coaxial to another at a distance <Dist>.
  • Create a Cylinder with 3 points.
  • Create a Cylinder by its axis and radius.
  • Create a cylinder by its circular base.
