Detailed Description

This class implements the algorithms used to create 2d circles tangent to an entity and centered on a point. The arguments of all construction methods are :

  • The qualified element for the tangency constrains (QualifiedCirc, Line, Point).
  • The center point Pcenter.
  • A real Tolerance. Tolerance is only used in the limits cases. For example : We want to create a circle tangent to an EnclosedCirc C1 with a tolerance Tolerance. If we did not used Tolerance it is impossible to find a solution in the the following case : Pcenter is outside C1. With Tolerance we will give a solution if the distance between C1 and Pcenter is lower than or equal Tolerance.
