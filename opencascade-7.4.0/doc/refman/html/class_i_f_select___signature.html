Detailed Description

Signature provides the basic service used by the classes SelectSignature and Counter (i.e. Name, Value), which is :

  • for an entity in a model, give a characteristic string, its signature This string has not to be unique in the model, but gives a value for such or such important feature. Exemples : Dynamic Type; Category; etc
