Detailed Description

This class implements the transfer of the Curve Entity from Geom To IGES. These can be : Curve . BoundedCurve.

  • BSplineCurve
  • BezierCurve
  • TrimmedCurve . Conic
  • Circle
  • Ellipse
  • Hyperbloa
  • Line
  • Parabola . OffsetCurve
