Detailed Description

template<class NumType, int Dimension, class DataType = Standard_Integer>
class BVH_BoxSet< NumType, Dimension, DataType >

Implements easy to use interfaces for adding the elements into BVH tree and its following construction. To make it more effective it is better to set the number of elements that are going to be added into BVH tree. For better efficiency on heavy data types it is recommended to use either BHV_IndexedBoxSet which uses indirect indexing for accessing the elements and their boxes or set the element to be an index of the real element in the application's internal data structures.

DimensionVector dimension
DataTypeType of elements on which the boxes are built
