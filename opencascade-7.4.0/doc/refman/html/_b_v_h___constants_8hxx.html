BVH_Constants_MaxTreeDepth 

The optimal tree depth. Should be in sync with maximum stack size while traversing the tree - don't pass the trees of greater depth to OCCT algorithms!

BVH_Constants_LeafNodeSizeSingle 

Leaf node size optimal for complex nodes, e.g. for upper-level BVH trees within multi-level structure (nodes point to another BVH trees).

BVH_Constants_LeafNodeSizeAverage 

Average leaf node size (4 primitive per leaf), optimal for average tree nodes.
