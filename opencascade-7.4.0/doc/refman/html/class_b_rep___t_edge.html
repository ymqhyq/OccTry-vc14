Detailed Description

The TEdge from BRep is inherited from the TEdge from TopoDS. It contains the geometric data.

The TEdge contains :

  • A tolerance.
  • A same parameter flag.
  • A same range flag.
  • A Degenerated flag.
  • A list of curve representation.
