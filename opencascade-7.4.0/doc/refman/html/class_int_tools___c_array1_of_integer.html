Creates an array sharing datas with an other. Example: Item tab[100]; CArray1OfItem thetab (tab[0],100);.

CArray1OfItem aArray1(100); CArray1OfItem anSharedArray1(aArray1.ChangeValue(0),aArray1.Length());
